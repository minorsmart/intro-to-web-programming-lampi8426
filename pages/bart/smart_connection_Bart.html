HTML & CSS

Bij het vak Smart connection is als eerste een website bouwen met HTML en CSS code behandeld. Ik had zelf helemaal geen ervaring met coderen van een website en had helemaal nooit gehoord van HTML en CSS.
Ik heb hiervan geleerd hoeveel werk er in een website bouwen zit op deze manier. Ik had zelf best wel moeite om het te snappen, het is een manier van denken.
Hierdoor kon ik wel veel leren en ik ben goed geholpen door klasgenoten als ik het niet goed snapte. We hebben tijdens deze les een standaard ingevulde website code gekregen en hiervan de kleur, lettertype en lettergrootte aangepast.
De opeenvolgende les ging hierop verder maar dan ook met CSS code en heeft het script verder stap voor stap uitgelegd, waardoor het voor mij veel helderder werd hoe het script in elkaar zat.
Onderstaand staat de link voor deze aangepaste pagina die wij hebben gebruikt in de eerste lessen.

API

Ik heb de lessen van API helaas gemist. Echter ben ik hier thuis wel mee aan de gang gegaan. Ik heb zelf moeten uitzoeken hoe dit in elkaar zit, dit was wel lastig maar uiteindelijk is het mij gelukt.
Ik vond dit interessant aangezien ik op diverse momenten al eens van API had gehoord maar nog niet wist wat het deed. Het kan dus informatie ophalen uit andere websites.

Samenvattend

Tijdens dit vak heb ik veel nieuwe dingen geleerd. HTML website bouwen, CSS code, API weersite, API NASA foto van de dag en AI. Deze lessen waren niet mijn favoriet,
echter was het wel heel interessant om te zien hoe alles in elkaar zit achter een website. Ik heb bij deze lessen veel gehad aan mijn ICT klasgenoten die mij hebben geholpen bij problemen.
Dit vak heeft mij veel nieuws gebracht en deze kennis is zeker nuttig om te begrijpen hoeveel werk er werkelijk zit in software ontwikkeling.

Competenties Smart Connection
Tijdens de lessen van Smart Connection is er gewerkt aan verschillende competenties. Alle competenties zijn terug te lezen via deze link.
Ik heb gewerkt aan de volgende competenties:
  • 1. Initiatief en betrokkenheid:
    1. Ik ben bij alle mogelijke lessen aanwezig geweest en ben actief betrokken geweest tijdens de lessen

  • 2. Meningen of adviezen zijn gebaseerd op gedegen onderzoek:
