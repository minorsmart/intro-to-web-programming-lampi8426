HTML pagina
Bij het vak Smart Connection zijn we begonnen met het bouwen van een website. We kregen les in het schrijven van HTML en CSS.
Ik vond het leuk om te zien hoe een website in elkaar zit, maar ik vond het ook best ingewikkeld. Ik heb van de lessen veel geleerd, ik kwam blanco binnen en ging met veel nieuwe kennis naar huis.
In de les was ik actief en heb ik zoveel mogelijk de CSS file bewerkt door beter te begrijpen wat er veranderd. Ik heb verschillende afbeeldingen toegevoegd, kleuren veranderd en tekst grootte aangepast.
De tweede les heeft mij veel gebracht doordat we stap voor stap het script bekeken en daarbij uitleg werd gegeven over de betekenissen van de codes.
Ik begrijp nu beter wat ik schrijf en waarom er bepaalde tekens in moeten staan om het te laten werken.
Hieronder is de link naar de Hello World pagina in HTML.

API
Vervolgens kregen we les over API. Een onderwerp dat ik nog niet kende. Ik vond het een best lastig onderdeel dus ben ik naast een IT student gaan zitten en hij heeft mij op weg geholpen.
Het was leuk om te zien als de website werkte en je de temperatuur van een plaats kan weergeven. De website bouwen met de NASA foto was iets ingewikkelder. Ik had alle stappen gevolgd van de opgegeven instructie, maar kwam er snel achter dat er typefouten en fouten in de verwijzing zaten.
Uiteindelijk heb ik dit kunnen oplossen en heb ik de style ook nog wat aangepast.

NASA picture of the day


We kregen ook les over Python programmeer taal en AI. Het was voor mij volledig nieuw en ik vond het best lastig.
We kregen theoretische uitleg en wat voorbeelden te zien. Ik heb tussendoor vragen gesteld aan klasgenoten om het beter te kunnen begrijpen.
In de laatste les hebben we de data van de ESP32 verzonden naar een cloud en daarvan grafieken gegenereerd. Ik heb veel geleerd in deze les, aangezien ik niet wist hoe zoiets werkte en wat erbij komt kijken om dit werkend te maken. Om een voorbeeld te noemen; Het koppelen van de WIFI en de juiste token en bucket heeft enorme zorgvuldigheid nodig, een kleine fout kan ervoor zorgen dat het script en Influx niet communiceren met elkaar.
Door codes op te zoeken en de betekenis te achterhalen heb ik de programmeer taal beter kunnen begrijpen en weet nu hoe een influxcloud koppelen werkt.


Competenties
Je toont veel initiatief en betrokkenheid. De lessen van Connection vond ik niet makkelijk en heb betrokkenheid getoond door vragen te stellen aan docenten en studenten. Door elke les aanwezig te zijn en actief deel te nemen aan de les, ben ik het beter gaan begrijpen.
Je meningen en adviezen zijn gebaseerd op gedegen onderzoek. De instructie van de NASA Picture Of The Day klopte niet helemaal en voor de problemen die ik tegen kwam heb ik onderzoek gedaan op internet. Ik heb andere instructies bekeken om te vergelijken wat er fout kon zijn en de foutmeldingen die ik kreeg heb ik opgezocht om erachter te komen wat de betekenis ervan is.
Je kiest ambitieuze leerdoelen en verkent disciplines buiten je eigen vakgebied. Ik heb disciplines verkent die volledig buiten mijn vakgebied zijn en waar ik totaal geen interesse voor had. Uiteindelijk vond ik het programmeren van HTML, CSS en JAVA erg leuk!
Je bent bewust bezig met je persoonlijke groei. Om de Hello world pagina en NASA opdracht te voltooien heb ik mijn kennis over een script schrijven enorm vergroot. Ik had vooraf geen kennis en door naast de lessen internet te raadplegen ben ik het gaan begrijpen.
Je gebruikt feedback en zelfreflectie om te leren. De Hello World pagina heb ik in de les laten controleren door Frank. Als ik de pagina live wilde draaien werd de Style niet meegenomen, Frank gaf mij de feedback dat ik de link in de HTLM pagina verkeerd had gebruikt waardoor het script de CSS niet kon vinden.
